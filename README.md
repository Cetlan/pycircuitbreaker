# PyCircuitBreaker

Python Implementation of the [Circuit Breaker Pattern](https://martinfowler.com/bliki/CircuitBreaker.html). Loosely based on [circuitbreaker](https://github.com/fabfuel/circuitbreaker) by Fabian Fuelling.
